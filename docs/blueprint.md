# **App Name**: TubeTrend AI

## Core Features:

- User Authentication: Secure user accounts with Clerk or NextAuth for login, registration, and session management.
- API Key Onboarding: A guided onboarding process where users can securely add their Gemini API key, YouTube Data API v3 key, and Google Trends API key.
- Dashboard: Main dashboard displaying channel stats, trending keywords, and AI-driven suggestions.
- Keyword Research: Tool for identifying trending keywords using Google Trends API.
- Competitor Analysis: Analyze competitor videos using the YouTube API to extract key performance metrics and strategies.
- AI Content Tools: Set of generative AI tools for title generation, description generation, tag generation, and script outline generation using the Gemini API. Each tool will incorporate user data when providing suggested content.
- YouTube Channel Analyzer: Tool that will show top videos and performance metrics. Information will come from user's YouTube channel.

## Style Guidelines:

- Primary color: Electric Indigo (#6F00FF) to convey innovation and intelligence.
- Background color: Very light gray (#F0F0F0), nearly white, to provide a clean and modern backdrop.
- Accent color: Bright cyan (#00FFFF) for interactive elements and highlights.
- Body and headline font: 'Inter', sans-serif, providing a modern, clean and versatile typeface for readability and a tech-focused aesthetic.
- Use a set of consistent, modern icons, optimized for clarity and easy recognition.
- Clean and structured layout with clear hierarchy to ensure an intuitive user experience.
- Subtle and smooth transitions and animations to enhance user engagement without being distracting.
